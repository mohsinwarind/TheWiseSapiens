Rules

1. Each individual participant must code their own project for the competition.

2. Participants are allowed to use AI assistance for guidance but must not directly copy-paste code.

3. The competition will apply Object-Oriented Programming (OOP) concepts.

4. Participants must submit their code on GitHub by the specified deadline.

5. Plagiarism or any form of cheating will result in immediate disqualification.

6. Participants must provide proper documentation and comments within their code to explain their logic and approach.

7. The code must be functional and should compile without errors.

8. The judges' decisions are final and binding.

9. Participants should be prepared to present and explain their code to the judges if required.

10. Feedback and scoring will be provided based on criteria such as code efficiency, creativity, adherence to OOP principles, and overall functionality.

11. The top 3 winners will be given a shoutout on the official page of the organization.

12. Each of the participant will be awarded a certificate of participation within 15 days of the following month.

13. A README file for explaning the code and a video showing the output of program is mandatory for submission.You can check this demo Virtual Mall to figure it out.

Frequently Asked Questions

What is CodeConquest?
CodeConquest is an exciting coding competition where Java developers from all over can showcase their skills, learn from peers, and push their coding abilities to the next level.
Who can participate?
Anyone with a passion for coding and an interest in challenging themselves can participate in CodeConquest. Whether you're a beginner or an experienced coder, our event offers a platform for growth and achievement.
How can I register for CodeConquest?
You only need to register youself while submission of code. We have provided the link to submission just a little bit up in the "Register" Section!
Is there a registration fee for participating in CodeConquest?
No, participation in CodeConquest is free for all. Anyone from any section can participate.
What are the judging criteria for CodeConquest?
Judging criteria typically include factors such as code efficiency, creativity, adherence to the competition rules, and overall functionality.
Are teams allowed to participate in CodeConquest?
CodeConquest is designed for individual participants. Collaboration between participants is not permitted.
When does CodeConquest start and end?
CodeConquest will start on June 19 and end on June 30. Make sure to mark your calendars!
Will there be certificate for participants of CodeConquest?
Yes, participants will be awarded a certificate of participation within 15 days of the following month.
Can I use any programming language for CodeConquest?
Almost Yes, CodeConquest requires the use of the Java programming language and adherence to Object-Oriented Programming (OOP) concepts however you can code your project in any language but using OOP concepts.
Is Uploading your project on Github mandatory?
Obviously Yes. You must need to upload your project file on Github by making a repository along with a README file for explaning the code.You can check this demo Virtual Mall to figure it out.
